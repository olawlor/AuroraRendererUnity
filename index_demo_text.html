An interactive renderer drawing Earth and the Aurora Borealis in 3D, built in the Unity game engine.

Keyboard shortcuts:

  • Mouse: look around
  • WASD: move horizontally
  • QZ: move up and down
  • Spacebar: toggle Earth rotation
  • Escape: exit fullscreen

The original volume datasets and rendering architecture are by: Dr. Jon Genetti, jdgenetti@alaska.edu

The Unity port, shader code, and GPU volume renderer are by: Dr. Orion Lawlor, lawlor@alaska.edu

Source code is available on GitHub.
